相似词接口使用说明：
接口统一访问前缀：http://IP:PORT
本服务访问地址：http://152.136.156.185:8090
1.短语相似度对比
  • 
    接口请求

    请求地址：/api/similarity/compare
    请求类型：支持GET以及POST

  • 
    请求参数
    {
    "sen1": "用户信息",
    "sen2": "新增用户"
}
  • 
    响应结果
    {
    "code": 200,
    "data": {
        "sen1": "用户信息",
        "sen2": "新增用户",
        "similar": "0.51"
    },
    "message": "操作成功"
}
  • 
    结果解释

    通过data.get('similar')获取sen1与sen2的相似度值

2.多短语两两相互对比
  • 
    接口请求

    请求地址：http://localhost:8099/api/similarity/mutual
    请求类型：支持GET以及POST

  • 
    请求参数
    {
    "mode": "map",
    "sens": "用户信息;新增用户;删除用户;新建客户"
}
  • 
    响应结果
    {
    "code": 200,
    "data": {
        "k-0-1": 0.512,
        "k-0-2": 0.537,
        "k-0-3": 0.459,
        "k-1-2": 0.513,
        "k-1-3": 0.934,
        "k-2-3": 0.465
    },
    "message": "操作成功"
}
  • 
    结果解释

    通过data.get('k-i-j')获取第i个词语与第j个词语的相似度值
